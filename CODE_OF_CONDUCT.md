# Code of Conduct

## Purpose

このコミュニティは、公共データ活用の実践知を共有する場です。  
参加者全員が安心して議論と実装に参加できる環境を維持します。

## Expected Behavior

- 相手を尊重し、建設的なフィードバックを行う
- 事実と根拠に基づいて議論する
- レビュー時はコードと仕様に集中し、人格攻撃をしない
- 不明点は断定せず質問として扱う

## Unacceptable Behavior

- ハラスメント、差別、侮辱、脅迫
- 悪意のある個人情報公開
- 議論の妨害や、継続的な攻撃的コミュニケーション
- 公共データの不正利用を助長する投稿

## Scope

この行動規範は以下に適用されます。

- GitHub Issues / Pull Requests / Discussions
- プロジェクトに関連する公開・非公開のコミュニケーション

## Reporting

問題を見つけた場合は、メンテナーへ連絡してください。  
公開の場での報告が難しい場合は、GitHub の非公開連絡手段（Security Advisory など）を優先してください。

## Enforcement

メンテナーは、行動規範違反に対して警告、投稿削除、アクセス制限などの対応を行うことがあります。  
重大または反復的な違反は、参加停止とします。
